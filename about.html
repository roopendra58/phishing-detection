Advancing Phishing Detection Through Machine Learning

A comprehensive research initiative combining Neural Architecture Search, Deep Learning models, and feature engineering to achieve state-of-the-art phishing URL detection with 99.2% accuracy.

Research Methodology

Our comprehensive approach combines feature engineering, machine learning models, and neural architecture search for optimal phishing detection.

Feature Engineering

Careful selection and extraction of 20 URL features spanning structural, content-based, and reputation metrics for comprehensive analysis.

Model Architecture

Implementation of 6 advanced DL models including DNN, FNN, GCN, Autoencoders, SAINT, and Neural Architecture Search (NAS).

Optimization

Hyperparameter tuning, cross-validation, and performance optimization using gradient descent and reinforcement learning techniques.

Phase 1

Data Collection & Preprocessing

Collected URL samples from legitimate and phishing sources. Implemented rigorous preprocessing including feature extraction, normalization, and dataset balancing.

Phase 2

Feature Selection & Engineering

Identified 20 key URL features using Information Gain and MRMR algorithms. Engineered features covering URL structure, hostname characteristics, and domain reputation metrics.

Phase 3

Model Development & Training

Implemented and trained 6 ML models. Used Neural Architecture Search to automatically discover optimal neural network architectures for maximum accuracy.

Phase 4

Evaluation & Deployment

Achieved 95% accuracy with NAS model. Implemented accurate prediction API and user-friendly web interface for researchers and cybersecurity professionals.
